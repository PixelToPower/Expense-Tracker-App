# Stock Buy/Sell Profit Calculator

A web-based calculator that analyzes stock price data and calculates potential profit based on user-defined inputs.

## What it does
Takes user input including:
- Dates and stock prices for a chosen number of days
- Holding time (limited to the number of days of price data entered)
- Quantity of stock

And returns:
- The minimum and maximum stock price, along with their respective dates
- Daily profit or loss compared to the first day's entry price
- Net profit for the specified holding period
